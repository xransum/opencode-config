# No Banner or Section Divider Comments

Do not add section divider or banner comments to code. This includes decorative
separators like:

```
# ---------------------------------------------------------------------------
# Helpers
# ---------------------------------------------------------------------------
```

or any variation using dashes, equals signs, hashes, or other repeated
characters to visually separate sections. These are considered noise -- good
code should be self-documenting through clear naming, structure, and docstrings
instead.
